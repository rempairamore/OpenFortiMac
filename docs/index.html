# features

Click once. You're in.

OpenFortiMac is openfortivpn with a menu bar icon and a Keychain entry. You get the good parts of the CLI client without a terminal sitting open all day, and without the weight of the official FortiClient that still manages to forget your password.

--one-click-connect
Click the menu bar icon, pick a server, you're in. Disconnect the same way.
--keychain-secured
Passwords live in the macOS Keychain. Not in a config file, not in an env var.
--multi-profile
Add as many servers as you need. Switch between them straight from the menu bar.
